# Simone — Design Context

合成自 `CLAUDE.md`，老鱼已验收 v1.0 → v1.2，v1.3.0 已提审 2026-04-23。

## Users

**谁**：对 AI 氛围音乐感兴趣的成年用户，iPhone，个人独处时聆听。
**场景**：都市夜晚、通勤、居家工作、深夜独处。非活跃交互——「选台就完了」，像真的电台，打开听，不是播放器。
**Job to be done**：让房间/耳朵里有一层恰到好处的背景声，不抢注意力但有质感。

## Brand Personality

**3 个词**：都市夜晚 / 温柔陪伴 / 克制带一点梦感（取自 CLAUDE.md 原话）
不是 "modern" 不是 "elegant" —— 是 **2am、一个人、耳机戴着、屋里没其他光源**。

**Voice**：
- 不卖力，不激情，不"酷"
- 像老朋友坐在副驾驶，不说话也舒服

## Aesthetic Direction

**Theme**：Dark（v1.2 Fog 钦定）— 深夜聆听场景。
**调色**：Fog City Nocturne · OKLCH 冷色，深夜雾都。底色 oklch(0.15 0.015 250)。
**Typography**：Unbounded（display）/ Fraunces（serif body）/ Archivo（small caps）—— 已 bundle 到 app 中。
**视觉比喻库**（v1.2 内部）：
- Fog 雾、Nocturne 夜曲、Vinyl 仪式物、City 都市骨架

**Anti-references**（决不要像）：
- Spotify/Apple Music — 过分「播放器感」
- 通用音乐 visualizer — EQ bar、neon 光柱、cyber 网格
- AI slop — cyan-on-dark、purple-to-blue gradient、glass card、border-left 彩条
- Flat icon 插画、toy-like emoji

## Design Principles

1. **物件感 > UI 感** — Visualizer 不是图表，是房间里的一件物（黑胶机、雾都天际线、窗上雨）。它存在是因为它美，不是因为它报告数据。
2. **沉浸优先** — 任何反馈、loading、对话框不主动打断用户（memory: feedback_no_break_immersion）。
3. **克制的温度** — 暖意点缀（黄铜、烛光、橙红 label），但底色永远是冷雾蓝。不花哨。
4. **侧光而非顶光** — 场景光从单侧来，大半在阴影里。不要平铺照明。
5. **动而非跳** — spectrum 驱动微动（几像素、几度），不要 bar 一样猛跳。
